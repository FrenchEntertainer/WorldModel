## Purpose of the project
The purpose of the project is to try to see if a Machine Learning Approach can be applied to what has been in the past the 
field of System Dynamics, ie if we can replace the explicit Dynamic Equations with pure statistical model, using for instance
Neural Networks.
As the approach seems to work on Physical Systems traditionnaly studies through ODE or PDE (see the progress made in Meteorology), 
there is no reason we should not be able to do it also for areas which have been studied under the lens of System Dynamics.
## Files
Dynamics-of-growth-in-a-finite-world : the original full description of the World3 model, the simulation underlying the report 
of the Club of Rome Limits to Growth.
World3_population : a notebook to describe and simulate the population module of the World3 model.
world3_population_output.csv : the output.
